ISTRUZIONI PER VILLA ELYSIA TROPEA

Questa cartella contiene tutti i file del sito web pronti per la pubblicazione o per la condivisione.

COSA FARE ADESSO:
1. Apri la cartella 'assets/images' e inserisci le foto reali della struttura rinominandole in questo modo:
   - 'hero-bg.jpg' -> Foto principale di copertina
   - 'about-garden.jpg' -> Foto per la sezione giardino/esterno
   - 'room-comfort.jpg' -> Foto della camera standard/comfort
   - 'room-balcony.jpg' -> Foto della camera con balcone

2. Per vedere l'anteprima locale del sito ti basta fare doppio clic sul file 'index.html'.

3. Per condividere questo lavoro con il proprietario, comprimi l'intera cartella principale in un archivio .zip e inviaglielo.
